# Key generation
```bash
rm -rf ./keys && mkdir -p ./keys/client ./keys/server
ssh-keygen -t rsa -q -N "" -f ./keys/client/ssh_key
ssh-keygen -t rsa -q -N "" -f ./keys/server/ssh_key
```
